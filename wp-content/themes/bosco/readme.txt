== Description ==

A personal blog theme with a responsive layout and beautiful typography.

= Quick Specs (all measurements in pixels) =

* The main column width is 520.
* The footer widget area widths are 205.
* Featured Images are 750 wide by unlimited height on both Sticky posts, and as header images on single posts and pages.
